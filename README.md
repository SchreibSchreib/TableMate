# TableMate

TableMate is a mobile application designed to help board game groups organize and manage their game nights. The app provides a central place for scheduling events, rotating hosts, suggesting games, and voting on what to play.

Planned features include viewing upcoming game nights, managing host rotation, collecting and voting on game suggestions, rating past events, and communicating with group members. Optional features include support for coordinating food preferences and group orders.

This project is being developed as part of a software engineering course and is currently in an early stage of development.
